# ionic pwa example

Simple example using ionic with sw, manifest and native components. 

  - Service works
  - media device
  - location

### Installation

Install the dependencies and devDependencies and start the server.

```sh
$ npm install 
$ ionic serve
```

For production

```sh
$ ionic build --prod
```
